PER INSTALLARE:
sudo apt install snapd
sudo snap install snapd
sudo snap install microk8s --classic

per aggiungere microk8s al path di sudo:
sudo visudo
aggiungere alla fine della riga "Default..": :/snap/bin e salvare.
